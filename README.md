# kevin_bacon

Trabalho para a disciplina SCC-0503 - Algoritmos e Estruturas de Dados II 

Consiste na montagem de um grafo com os dados de um arquivo para verificar o número de Kevin Bacon de um determinado ator / pessoa. 
O número de Kevin Bacon consiste no menor caminho para ligar um ator e o Kevin Bacon através de participações em filmes.
Além disso, o programa fornece média e desvio padrão do número Kb dos atores contidos no arquivo.
